# **App Name**: Data Insights Portfolio

## Core Features:

- Interactive Project Display: Showcase data analysis projects with interactive charts and summaries in a modal view.
- Resume Builder: Use user entered experience data to help generate the resume.
- Skills and Services Highlight: Clearly present skills and data analysis services offered with concise descriptions.
- Key Performance Indicators (KPIs): Highlight impactful KPIs achieved, quantified with real metrics from past projects.
- Blog Insights Teaser: Feature previews of data analysis insights and articles.
- Contact Form: Allow visitors to easily contact the Data Analyst through a submission form.

## Style Guidelines:

- Primary color: Cool blue (#468B97) for calls to action and links, reflecting expertise and reliability.
- Background color: Light gray (#F5F5F5) for a clean, airy, neutral backdrop.
- Accent color: Soft cyan (#64CCC5), to give some lightness, used sparingly to draw attention.
- Body and headline font: 'Inter', a sans-serif font providing a modern, neutral, and readable experience.
- Simple, minimalist icons to represent data analysis skills and services.
- Clean, card-based sections with ample whitespace for readability and a professional appearance.
- Subtle hover transitions on cards and buttons to provide interactivity without overwhelming the user.
